# Accessibility

Making AI coding tools work with assistive technology.

This section covers screen reader configuration, keyboard-only workflows, and platform-specific setup for using Copilot, Claude Code, and Codex with assistive tech.

!!! accessibility "This section is for everyone"

    Accessibility is not a feature; it is a requirement. These docs are written
    for users of assistive technology AND for sighted developers who want to
    understand how their tools work for others.
